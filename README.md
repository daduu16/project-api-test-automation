# API Test Automation Project

This project demonstrates REST API test automation using Java and RestAssured.

## Technologies
- Java
- RestAssured
- JUnit 5
- Maven
- Allure Report

## Project Structure
src/test/java
   ├─ client
   ├─ models
   ├─ tests
   
## Run Tests
mvn clean test

## Generate Allure Report
allure serve target/allure-results
